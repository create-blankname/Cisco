## Лабороторная работа №2. DHCP

**DHCP** ( Dynamic Host Configuration Protocol – протокол динамической настройки узла ) – сетевой протокол прикладного уровня, это протокол автоматической настройки, используемый в IP-сетях. DHCP-сервер может назначать каждому клиенту IP-адрес и информировать клиентов об IP-адресе DNS-сервера и IP-адресе шлюза по умолчанию.

**DHCP на маршрутизаторе**

```
conf t
ip dhcp pool computers - создать пул DHCP с названием computers
network 192.168.140.0 255.255.255.0 - задать IP адреса и маску сети  


```
